What is variable in JS?

Ans. -
- Variables are containers for storing information. (simply a name of storage location)
- A variable is a “named storage” for data.

Type of Variable

Ans. -
- There are three types of variables in JavaScript.

  1. var
  2. let
  3. const

- There are two types of scope variables in JavaScript.
  1. global
  2. local/block

describe variable with scope variable

  1. var is global scope
  2. let is local/block scope
  3. const is local/block scope
