1 . Globle Scope (var)

- A JavaScript global variable is accessible from any function.
- Global scope contains all of the things defined outside of all code blocks.

var:

eg.1

var color = "pink";
{
var color = "blue";
console.log(color);
}
console.log(color);

ans :- blue, blue

eg.2

var color = "pink";
{
console.log(color);
}
console.log(color);

ans :- pink, pink

eg.3

var color = "blue";
{
var color = "blue";
console.log(color);
}
console.log(color);

ans :- blue, blue

2 . local Scope (let , const)

- Local scope contains things defined inside code blocks.
- A local variable has local scope.
- A local variable is only accessible where itâ€™s declared.

let:

eg.1

{
let name = "Khyati";
console.log(name);
}

ans :- Khyati

eg.2

{
console.log(name);
}

eg.3
